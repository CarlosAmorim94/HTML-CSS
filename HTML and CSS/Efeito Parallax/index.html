Efeito Parallax

Rolagem parallax é uma técnica de web design onde o background se movimenta em um ritmo mais lento que o primeiro plano. ... Com os avanços em CSS e HTML, os efeitos parallax evoluíram posteriormente, para o mundo do web design como o conhecemos hoje.

Com a ajuda de várias camadas que são sobrepostas umas às outras, é possível criar um efeito tridimensional artificial, podendo aplicar em ícones, imagens e outros elementos.
